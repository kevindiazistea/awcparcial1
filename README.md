# Parcial 1 - Apliaciones Web Cliente

El objetivo del parcial 1 es construir una vista de administración con un menú lateral, un botón hamburguesa para expandir y contraer el menú con un contenido dinámico que cambia según la opción seleccionada.
Con la idea de simular una interfaz de un panel administrativo.
  
[GitHub Kevin Diaz - Parcial 1](https://kevindiazistea.github.io/awcparcial1/#)

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

## Estructura del proyecto

- index.html           → Estructura del proyecto
- /css/styles.css      → Estilos del proyecto
- /imagenes            → Imagnes de las secciones
- /js/script.js        → Funciones del proyecto (Menu hamburguesa)

## La web nos debe permitir

- *Ver una vista de administración ordenada*
- Barra superior fija con el título de administración y opciones de usuario (idioma y logout)
- Menú lateral izquierdo con íconos y textos descriptivos para cada sección.
- Zona central de contenido, donde se muestra lo que el usuario selecciona.
- *Al hacer clic en el botón hamburguesa*
- El menú se expande o se contrae.
- Cuando se contrae, el texto de cada opción pasa debajo del ícono.
